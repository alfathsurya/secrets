# Enum-Web-Bypass-Upload

## Alternate Extensions

```
php: .pht, .phpt, .phtml, .php, .php3, .php4, .php5, .php6, .php7, .inc
asp: asp, .aspx
perl: .pl, .pm, .cgi, .lib
jsp: .jsp, .jspx, .jsw, .jsv, and .jspf
Coldfusion: .cfm, .cfml, .cfc, .dbm
```

## Change case

```
file.PHP
file.AsPx
```

## Disguise filename

```
file.php.jpg
file.php;.jpg
```

## Use null-byte

```
file.php%0-0.gif (remove the "-" for null byte)
```
